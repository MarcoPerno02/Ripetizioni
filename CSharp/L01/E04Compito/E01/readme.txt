ESERCIZIO: Mini Quiz a risposta multipla

Obiettivo:
Creare un'app WinForms in C# che mostra una domanda con tre risposte.
L’utente seleziona una risposta e clicca un bottone per sapere se è corretta.

Interfaccia richiesta:
- Label: per mostrare la domanda
- RadioButton (3): opzioni di risposta
- Bottone "Verifica risposta"
- Label: per mostrare il risultato (corretto/sbagliato)

Funzionalità:
- Solo una risposta è corretta
- Verifica con confronto di quale RadioButton è selezionato
- Mostra risultato in una Label (verde per corretto, rosso per sbagliato)
- Se nessuna risposta è selezionata, mostra un messaggio d’errore

Regole:
- Le risposte e la domanda sono fisse, non servono classi o file esterni
- Non è necessario caricare domande da file o creare un sistema di punteggio

Estensione facoltativa:
- Aggiungere più domande (es. con ComboBox per selezionare la domanda)
